I am currently developing an app called Psyke, an Android and desktop companion app built in Python using Kivy/KivyMD. The app includes features for tracking sanity, fears, madness, wounds, and spells, with local JSON save support, session state management, and Android-specific handling such as back button support and APK build configuration through Buildozer and GitHub Actions. My Play Console experience is currently limited because I am preparing this as my first release, but I do have hands-on Android development experience through building, testing, and packaging this project for Android, as well as managing the app structure, UI, persistence, and gameplay logic myself.
